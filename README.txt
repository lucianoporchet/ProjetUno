====================================================
       Projet de session d'automne Equipe Uno
====================================================
                   Space Escape
====================================================



====================================================
		      Contrôles
====================================================
W : Avancer
S : Reculer
A/D : Roulis
Shift/Espace : Roulis

Souris : Orientation de la caméra (détermine la
direction vers laquelle on avance/recule)

====================================================
			Notes
====================================================
Le seul moyen de fermer le jeu pour le moment est un
ALT+F4.
